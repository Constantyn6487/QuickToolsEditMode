 <div align="center"><h1> 🛠️ Quick Tools Edit Mode (v2.0) 🛠️ </h1>

![description](https://github.com/Constantyn6487/link/blob/main/preview170eng1.png?raw)

</div>

**QuickToolsEditMode** is a compact add-on for Blender 4.0+ that brings mission-critical Object Mode operations directly into the Edit Mode context menu (RMB). No more switching back and forth to Object Mode just to reset your scale or apply rotation.

## ✨ Key Features
- **Apply Transforms in Edit Mode**: Reset **Scale** and **Rotation** instantly without leaving Edit Mode.
- **Fast Weights (0.0, 0.5, 1.0)**: Set **Edge Bevel Weight** and **Edge Crease** with a single click.
- **Shading Management**: Quick access to **Shade Smooth** and **Shade Flat** toggles.
- **Smart Icons**: Full support for new **Blender 4.0 - 5.0** icons with automatic fallback to classic icons in older versions.

## 🚀 Installation
1. Download this repository as a `.zip` file.
2. In Blender, go to **Edit > Preferences > Get Extensions**.
3. Click the arrow icon in the top-right corner -> **Install from Disk**.
4. Select the downloaded ZIP archive.

## 🖱️ How to Use
While in **Edit Mode**, simply **Right-Click (RMB)**. You will find the **Quick Panel** at the top of the context menu, containing all the tools.

## 🧱 Possible errors: 
- **Double-clicking on the 0.5 button no longer changes the weight to 1**. The weights have become fixed, the weight will remain 0.5.
- **Now the weight is set to 0.5 if the weight is 1. When you press weight 0.5, the weight will become 0.5**.

<div align="center"><h1> 🛠️ Quick Tools Edit Mode (v2.0) 🛠️ </h1>

![description](https://github.com/Constantyn6487/link/blob/main/preview170rus.png?raw)

</div>

**QuickToolsEditMode** — это компактный аддон для Blender 4.0+, который переносит критически важные операции объектного режима (Object Mode) напрямую в контекстное меню режима редактирования (Edit Mode, RMB). Больше не нужно переключаться туда-сюда в объектный режим только для того, чтобы сбросить масштаб или применить вращение. 

## ✨ Ключевые особенности
- **Применение трансформаций в режиме редактирования**: Мгновенно сбрасывайте **Scale** и **Rotation**, не выходя из режима редактирования.
- **Быстрые веса (0.0, 0.5, 1.0)**: Устанавливайте **Edge Bevel Weight** и **Edge Crease** одним кликом.
- **Управление шейдингом**: Быстрый доступ к переключателям **Shade Smooth** и **Shade Flat**.
- **Умные иконки**: Полная поддержка новых иконок **Blender 4.0 - 5.0** с автоматическим откатом к классическим иконкам в старых версиях.

## 🚀 Установка
1. Скачайте этот репозиторий в виде `.zip` файла.
2. В Blender перейдите в **Edit > Preferences > Get Extensions**.
3. Нажмите на иконку стрелки в правом верхнем углу -> **Install from Disk**.
4. Выберите скачанный ZIP-архив.

## 🖱️ Как использовать
Находясь в **режиме редактирования**, просто нажмите **правую кнопку мыши (ЛКМ)**. Вы найдете **Quick Panel** в верхней части контекстного меню, содержащую все инструменты.

## 🧱 Возможные ошибки:
- **Двойной клик по кнопке 0.5 больше не меняет вес на 1**. Веса стали фиксированными, вес останется 0,5.
- **Теперь вес устанавливается на 0.5, если вес равен 1. При нажатии вес 0,5 вес станет 0,5**.
