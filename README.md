 <div align="center"><h1> 🛠️ Quick Tools Edit Mode (v1.0) 🛠️ </h1>

![description](https://github.com/Constantyn6487/link/blob/main/preview%201.0%20-%202.0%20Eng.png?raw=true)

</div>

**QuickToolsEditMode** is a compact add-on for Blender 3.6/5.0+, that brings mission-critical Object Mode operations directly into the Edit Mode context menu (RMB). No more switching back and forth to Object Mode just to reset your scale or apply rotation.

## ✨ Key Features
- **Apply Transforms in Edit Mode**: Reset **Scale** and **Rotation** instantly without leaving Edit Mode.
- **Fast Weights (0.0, 0.5, 1.0)**: Set **Edge Bevel Weight** and **Edge Crease** with a single click.
- **Shading Management**: Quick access to **Shade Smooth** and **Shade Flat** toggles.
- **Smart Icons**: Full support for new **Blender** icons with automatic fallback to classic icons in older versions.
- **Russian Interface**: Support for Russian UI and tooltips.
- **No support for version updates**: No auto‑update panel for new versions.

## 🚀 Installation
1. Download this repository as a `.zip` file.
2. In Blender, go to **Edit > Preferences > Add-ons**.
3. Click the arrow icon in the top-right corner -> **Install from Disk**.
4. Select the downloaded ZIP archive.

## 🖱️ How to Use
While in **Edit Mode**, simply click the **right mouse button (LMB)**. You will find **QuickToolsEditMode** at the top of the context menu, containing all the tools. Everything is also available in the **N‑Panel**.

## 🧱 Possible errors: 
- **Double click on button 0.5 changes weight to 1**.
- **Set to 0.5 if weight equals 1, when pressing 0.5 weight will remain 1**.

<div align="center"><h1> 🛠️ Quick Tools Edit Mode (v1.0) 🛠️ </h1>

![description](https://github.com/Constantyn6487/link/blob/main/preview%201.0%20-%202.0%20%20Rus.png?raw=true)

</div>

**QuickToolsEditMode** — это компактный аддон для Blender 3.6/5.0+, который переносит критически важные операции объектного режима (Object Mode) напрямую в контекстное меню режима редактирования (Edit Mode, RMB). Больше не нужно переключаться туда-сюда в объектный режим только для того, чтобы сбросить масштаб или применить вращение. 

## ✨ Ключевые особенности
- **Применение трансформаций в режиме редактирования**: Мгновенно сбрасывайте **Scale** и **Rotation**, не выходя из режима редактирования.
- **Быстрые веса (0.0, 0.5, 1.0)**: Устанавливайте **Edge Bevel Weight** и **Edge Crease** одним кликом.
- **Управление шейдингом**: Быстрый доступ к переключателям **Shade Smooth** и **Shade Flat**.
- **Умные иконки**: Полная поддержка новых иконок **Blender** с автоматическим откатом к классическим иконкам в старых версиях.
- **Руссий Интерфейс**: Поддержка Русского интерфейса и подсказок.
- **Нет поддержки обновления версий**: Отсутствует панель автообновления новых версий.

## 🚀 Установка
1. Скачайте этот репозиторий в виде `.zip` файла.
2. В Blender перейдите в **Edit > Preferences > Add-ons**.
3. Нажмите на иконку стрелки в правом верхнем углу / и выберите -> **Install from Disk**.
4. Выберите скачанный ZIP-архив.

## 🖱️ Как использовать
Находясь в **режиме редактирования**, просто нажмите **правую кнопку мыши (ЛКМ)**. Вы найдете **QuickToolsEditMode** в верхней части контекстного меню, содержащую все инструменты. Все также есть и в N-Panel.

## 🧱 Возможные ошибки:
- **Двойной клик по кнопке 0.5 меняет вес на 1**.
- **Если вес равен 1, при нажатии 0,5 вес останется 1**.
