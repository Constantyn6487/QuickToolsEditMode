**QuickToolsEditMode** is a compact add-on for Blender 4.2+ that brings mission-critical Object Mode operations directly into the Edit Mode context menu (RMB). No more switching back and forth to Object Mode just to reset your scale or apply rotation.

## âœ¨ Key Features
- **Apply Transforms in Edit Mode**: Reset **Scale** and **Rotation** instantly without leaving Edit Mode.
- **Fast Weights (0.0, 0.5, 1.0)**: Set **Edge Bevel Weight** and **Edge Crease** with a single click.
- **Shading Management**: Quick access to **Shade Smooth** and **Shade Flat** toggles.
- **Smart Icons**: Full support for new **Blender 5.0** icons with automatic fallback to classic icons in older versions.
